We used these 4 algorithms in Our dataSet


Decision Tree Algorithm

use tree representation. Each internal node of the tree corresponds to an attribute, and each leaf node corresponds to a class label.

  
Random Forest Algorithm

combine tree predictors where each tree depends on the values of a random vector sampled independently with the same distribution for all trees in the forest.

  
Naive Bayess Algorithm

are simple, probabilistic classifiers that use Bayes' Theorem.

  
K_neighbors Algorithm

the input consists of the k closest training examples in the feature space. The output depends on whether k-NN is used for classification
